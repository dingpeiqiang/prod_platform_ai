## 今天完成的工作

### 修复编辑器条件分支卡片锚点与分支不匹配问题

**问题描述：**
编辑器中条件分支卡片的锚点（连接点）与分支不匹配

**根本原因：**
1. `calculateHandlePositions` 函数中的选择器 `.branch-condition-item:nth-child(${index + 1})` 无法正确找到对应的分支项
2. 当找不到箭头元素时，默认计算的位置分布与实际箭头位置不一致
3. 缺少对节点尺寸变化的监听，导致缩放后锚点位置不更新

**修复内容：**
1. 改用 `querySelectorAll('.branch-condition-item')` 直接获取所有分支项，通过索引精确定位
2. 修正默认位置计算逻辑，使其与实际箭头位置保持一致
3. 添加 `ResizeObserver` 监听节点尺寸变化，实时更新锚点位置

**修改文件：**
- `frontend/src/components/workflow-editor/nodes/ConditionNode.vue`
  - 第 460-501 行：`calculateHandlePositions` 函数
  - 第 521-534 行：`onMounted` 钩子（添加 ResizeObserver）
