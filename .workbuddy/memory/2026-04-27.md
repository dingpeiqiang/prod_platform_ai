## 新增管理模块（Admin Module）

### 后端（FastAPI）

**新增文件：**
- `backend/app/schemas/admin.py` - 管理模块 Pydantic Schema（表单本体 CRUD + 场景映射 + 统计）
- `backend/app/services/admin_service.py` - 管理 Service 层（读写 config/ontologies/*.json + config/scenes/scene_mapping.json，热重载）
- `backend/app/api/admin.py` - 管理路由（前缀 `/api/v1/admin`）

**已修改文件：**
- `backend/app/main.py` - 注册 admin_router，日志模块加入 admin_api、admin_service

**API 端点：**
- GET  /api/v1/admin/stats              — 系统统计概览
- GET/POST/PUT/DELETE /api/v1/admin/ontologies/{code}   — 表单 CRUD
- GET/PUT /api/v1/admin/scenes          — 场景关键词管理
- GET /api/v1/admin/form-instances      — 提交记录分页查询

### 前端（Vue3 + ElementPlus）

**新增文件（frontend/src/components/admin/）：**
- `AdminPanel.vue`    - 管理面板主组件（左侧菜单 + 右侧内容区）
- `AdminDashboard.vue` - 统计概览（卡片 + 表单列表预览）
- `FormManager.vue`  - 表单配置管理（列表/新增/编辑/删除/预览）
- `OntologyEditor.vue` - 可视化字段设计器（支持实体分组、字段增删、类型/选项配置、JSON 预览）
- `SceneManager.vue` - 场景关键词配置（tag 增删、优先级）
- `InstanceList.vue` - 提交记录分页查询

**已修改文件：**
- `frontend/src/App.vue` - 增加 mode 切换（chat/admin）、右上角「管理」按钮入口

### 架构特点
- 表单配置以文件方式存储在 `config/ontologies/`，修改后自动热重载（无需重启服务）
- 场景关键词同样存储在 `config/scenes/scene_mapping.json`，支持热重载
- 前端采用组件条件渲染切换，无需 Vue Router

## AI 对话式配置助手（Admin AI Chat）

**需求**：通过聊天窗口 + AI 能力，高效完成新业务表单配置，替代手动填表单字段。

### 后端新增文件

- `backend/app/services/admin_ai_service.py` - AI 配置核心服务
  - `AdminAiService.chat(messages, context)` - 多轮对话，通过 LLM 生成 ontology JSON 配置
  - `AdminAiService.generate_scene_keywords()` - AI 自动生成场景关键词
  - 内置配置格式校验 `_validate_ontology()`，响应包含 validationErrors
- `backend/app/api/admin_ai.py` - AI 配置 API 路由（前缀 `/api/v1/admin/ai`）
  - POST `/api/v1/admin/ai/chat`     - 多轮对话生成配置
  - POST `/api/v1/admin/ai/save`     - 保存 AI 生成的表单配置（支持 overwrite）
  - POST `/api/v1/admin/ai/keywords` - AI 自动生成关键词
- `backend/app/main.py` - 注册 admin_ai_router

### 前端新增文件

- `frontend/src/components/admin/AiConfigChat.vue` - AI 对话配置助手组件
  - 左：对话区（自然语言 → AI 回复 + 配置卡片预览 + 快捷建议）
  - 右：字段预览区（实体结构 + 字段类型标签 + 原始 JSON 折叠展示）
  - 保存弹窗：确认保存 + 同时生成关键词选项
- `frontend/src/components/admin/AdminPanel.vue` - 新增「AI 配置助手」菜单项（默认选中），保存后自动跳转到表单列表

### 工作流程

用户输入业务描述 → AI 生成 ontology JSON → 右侧预览字段结构 → 点击「保存配置」→ 同步生成场景关键词 → 跳转至表单列表查看
