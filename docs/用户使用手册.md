## 1. 快速开始

### 1.1 系统启动

#### Windows快速启动
1. **启动后端**：双击 `start-backend.bat`
2. **启动前端**：双击 `start-frontend.bat`

#### 手动启动
```bash
# 启动后端
cd backend
pip install -r requirements.txt
python -m app.main

# 启动前端（新终端）
cd frontend
npm install
npm run dev
```

### 1.2 访问应用

- **前端地址**：http://localhost:5173
- **后端API文档**：http://localhost:8000/docs

## 2. 聊天助手使用指南

### 2.1 与AI对话

#### 自然聊天
你可以像和真人对话一样和AI聊天：

```
用户：你好
AI：你好！我是AI智能助手。我可以帮你填写各种表单...

用户：你能做什么？
AI：我可以帮你：1. 生成和填写表单...
```

#### 流式推理过程
当你发送消息后，你会看到AI的**实时思考过程**：

```
🔍 正在分析用户输入...
📝 用户消息：帮我填一个请假申请
🎯 识别到表单需求
📋 匹配到表单：请假申请 (leave)
✅ 准备生成表单...
```

### 2.2 生成表单

#### 方式一：自然语言生成

在聊天窗口中输入类似以下内容：

**示例1：生成销售订单**
```
帮我填一个销售订单，客户姓名张三，客户电话13800138000，订单金额5000元
```

**示例2：生成请假申请**
```
我要请假，姓名李四，请假类型病假
```

**示例3：生成费用报销**
```
报销费用，费用类型差旅费，金额2000元
```

#### 方式二：快捷按钮
点击下方的快捷按钮快速生成常用表单：
- 销售订单
- 请假申请
- 费用报销
- 帮助

### 2.3 填写表单

#### 推荐值功能
- 当你点击输入框时，会显示历史推荐值
- 推荐值来自你之前提交过的数据
- 使用频次越高、时间越近的值排名越靠前

#### 默认值填充
- 从你输入的自然语言中提取到的值会自动填充
- 如果没有提取到值，会使用推荐值的第1个作为默认值
- 你可以随时修改这些默认值

#### 字段类型说明

| 字段类型 | 说明 |
|----------|------|
| **input** | 单行文本输入 |
| **textarea** | 多行文本输入 |
| **number** | 数字输入 |
| **date** | 日期选择 |
| **select** | 下拉选择 |

### 2.4 提交表单

1. 填写完所有必填字段后
2. 点击「提交」按钮
3. 系统会保存你的数据，这些数据会成为后续的推荐值
4. 提交后会自动切回聊天模式

## 3. 功能特性说明

### 3.1 通用聊天助手

聊天窗口是一个**通用的AI助手**，不仅能帮你填表单，还能：
- 回答你的问题
- 和你聊天对话
- 智能理解你的需求
- 基于业务本体理解

### 3.2 智能意图识别

系统会自动识别你想填写什么表单：

| 关键词 | 识别出的表单 |
|--------|-------------|
| 销售订单、销售、订单、客户 | 销售订单 |
| 请假、请假申请、休假 | 请假申请 |
| 报销、费用报销、报销申请 | 费用报销 |

**AI增强识别**（如果配置了LLM）：
- 基于完整的本体信息进行更准确的识别
- 理解更复杂的表达
- 提供置信度和推理说明

### 3.3 智能字段提取

系统会从你的自然语言中自动提取字段值：

**支持的表达方式：**
```
字段名是值
字段名为值
字段名：值
字段名: 值
字段名 值
```

**示例：**
```
客户姓名是张三
订单金额5000元
请假类型: 病假
```

### 3.4 历史数据推荐

#### 推荐值来源
1. **你自己提交过的数据**（优先级最高）
2. **所有用户提交过的数据**
3. **系统预设的静态推荐**（兜底）

#### 推荐值排序规则
- 使用频次越高，排名越靠前
- 你自己使用过的值，排名更靠前
- 最近使用过的值，排名更靠前

### 3.5 流式推理过程

当你发送消息时，你会看到AI的实时思考过程：

**推理步骤示例：**
```
🔍 正在分析用户输入...
📝 用户消息：帮我填一个请假申请
🎯 识别到表单需求
📋 匹配到表单：请假申请 (leave)
✅ 准备生成表单...
```

**步骤说明：**
- 💭 思考步骤：AI正在分析和处理
- ✅ 结果步骤：最终的结论或回复

### 3.6 表单校验

系统支持自动校验：

**内置校验规则：**
- 长度校验（最小长度、最大长度）
- 数值范围（最小值、最大值）
- 格式校验（邮箱、手机号、身份证、URL）
- 日期范围（最小日期、最大日期）
- 枚举校验（只能选择指定选项）
- 正则表达式自定义校验

**校验失败：**
提交时如果校验失败，会显示错误信息，你可以修改后重新提交。

## 4. 常见问题

### 4.1 聊天相关

**Q: AI可以和我聊天吗？**
A: 可以！聊天窗口是一个通用的AI助手，不仅能帮你填表单，还能回答问题、聊天对话。

**Q: 什么是流式推理过程？**
A: 这是让你看到AI思考过程的功能。当你发送消息后，你会看到AI如何一步步分析你的请求。

### 4.2 表单生成相关

**Q: 系统没有识别出我想填的表单？**
A: 你可以尝试更明确地表达，例如：
```
生成一个请假申请表单
```

**Q: 系统提取的字段值不对？**
A: 没关系，你可以在表单中手动修改。同时，你的修改会帮助系统在未来更准确地提取。

### 4.3 推荐值相关

**Q: 推荐值里没有我想要的？**
A: 推荐值是基于历史数据生成的。你可以手动输入，提交后下次就会出现在推荐值里了。

**Q: 如何让某些值不再出现在推荐里？**
A: 目前系统会自动记录所有提交的值。如果有特定需求，可以联系管理员。

### 4.4 其他问题

**Q: 表单提交后可以修改吗？**
A: 目前版本不支持修改已提交的表单。如需修改，请重新生成并提交。

**Q: 我的数据安全吗？**
A: 你的数据存储在本地数据库中，不会上传到任何第三方服务。

**Q: 如何启用AI增强功能？**
A: AI增强功能需要配置LLM API Key。请参考维护手册进行配置。
