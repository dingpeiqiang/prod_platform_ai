# 工作流编辑器连线功能说明

## 功能概述

工作流编辑器支持通过拖拽方式在节点之间创建连接线，实现工作流的流程定义。

## 使用方法

### 1. 基本连线操作

1. **开始连线**：将鼠标悬停在节点的连接点（Handle）上
   - 左侧的连接点是**输入点**（target，紫色）
   - 右侧的连接点是**输出点**（source，蓝色）

2. **智能吸附**：当鼠标靠近连接点 30px 范围内时
   - 连接点会自动高亮并放大
   - 显示旋转的引导圆环
   - 底部出现“已锁定连接点”提示
   - 此时点击即可快速选中该连接点

3. **拖拽连线**：点击连接点并拖拽到目标节点的连接点
   - 从源节点的**输出点**（右侧）拖拽
   - 到目标节点的**输入点**（左侧）
   - 智能吸附会帮助你准确连接到目标点

4. **完成连线**：释放鼠标，连接线即创建成功

### 2. 连接点说明

不同类型的节点有不同的连接点：

- **开始节点（Start）**：只有输出点（右侧），作为工作流起点
- **结束节点（End）**：只有输入点（左侧），作为工作流终点
- **普通节点**：既有输入点也有输出点，可以串联在工作流中
- **条件节点（Condition）**：有多个输出点（true/false分支）
- **循环节点（Loop）**：有多个输出点（body/end）

### 3. 连线规则

- ✅ 可以从 source 连接到 target
- ❌ 不能从 target 连接到 target
- ❌ 不能从 source 连接到 source
- ❌ 不能创建循环连接（A→B→A）
- ⚠️ 一个 target 只能有一个输入连接
- ✅ 一个 source 可以有多个输出连接

### 4. 删除连线

- 点击选中连接线（线会变成蓝色）
- 按 `Delete` 或 `Backspace` 键删除

### 5. 视觉反馈

- **默认状态**：
  - 输入点（左侧）：紫色圆点
  - 输出点（右侧）：蓝色圆点
  - 连接线：灰色线条
  
- **悬停状态**：
  - 连接点放大 1.4 倍，颜色加深
  - 节点显示淡蓝色边框提示
  - 连接线加粗并变色
  
- **连接中**：绿色脉动动画高亮显示
- **有效连接**：绿色稳定显示
- **无效连接**：红色抖动提示
- **选中状态**：蓝色加粗线条带发光效果

## 常见问题

### Q1: 无法看到连接点？

**解决方案**：
- 确保节点已正确加载
- 检查浏览器缩放比例是否正常
- 刷新页面重试

### Q2: 拖拽时没有反应？

**解决方案**：
- 确认是从正确的连接点开始拖拽（source → target）
- 检查是否启用了 JavaScript
- 查看浏览器控制台是否有错误信息

### Q3: 连线后看不到线条？

**解决方案**：
- 检查两个节点是否都在画布可见区域
- 尝试缩小视图查看完整工作流
- 刷新页面重新加载

### Q4: 无法删除连线？

**解决方案**：
- 先点击选中连线（变蓝表示选中）
- 确保焦点在画布上（点击一下画布空白处）
- 然后按 Delete 键

## 技术实现

### VueFlow 配置

```javascript
<VueFlow
  :nodes-connectable="true"
  :edges-connectable="true"
  :connect-on-drag="true"
  @connect="onConnect"
>
```

### 连接处理

```javascript
const onConnect = (params) => {
  saveHistory();
  addEdges(params);
  markDirty();
};
```

### Handle 样式

连接点使用 VueFlow 的 Handle 组件：

```vue
<Handle type="source" :position="Position.Right" />
<Handle type="target" :position="Position.Left" />
```

## 最佳实践

1. **从左到右布局**：建议按照工作流执行顺序从左到右排列节点
2. **避免交叉**：尽量减少连线的交叉，保持工作流清晰
3. **合理使用对齐**：使用工具栏的对齐功能整理节点位置
4. **定期保存**：养成经常保存工作流的习惯
5. **使用模板**：对于常见的工作流模式，可以使用快速模板

## 快捷键

- `Delete` / `Backspace`：删除选中的连线或节点
- `Ctrl + Z`：撤销操作
- `Ctrl + Y`：重做操作
- `Ctrl + S`：保存工作流
