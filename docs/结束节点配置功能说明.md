# 结束节点配置功能说明

## 功能概述

为工作流编辑器的结束节点添加了完整的配置面板,支持以下功能:

### 1. 顶部标题栏
- **左侧**: 黑色加粗标题"编辑工作流"
- **右侧**: 关闭按钮(×),浅灰色图标,点击关闭配置弹窗

### 2. 节点标识区
- **紫色方形图标**: 内含白色小旗帜,是结束节点的专属标识
- **输入框**: 显示"结束节点",可自定义节点名称
- **说明文字**: "工作流的最终节点,用于返回工作流运行后的结果"

### 3. 选择回答模式(下拉选择)
- **标签**: "选择回答模式"
- **下拉选项**:
  - 文本 (默认)
  - JSON
  - Markdown
  - HTML

### 4. 「输出」配置区(可折叠面板)
- **区域标题**: "输出",带向下箭头图标(展开状态)
- **帮助图标**: 带圆圈的 i 图标
- **添加按钮**: 蓝色 + 号按钮,添加新的输出参数

**参数配置表格**:
- **表头**: 参数名 | 参数值 | (操作)
- **每行配置项**:
  - 左侧输入框: 定义输出变量名称
  - 中间参数值配置:
    - 第一个下拉框: 引用/常量/表达式
    - 第二个下拉框/输入框: 根据类型选择具体值
  - 右侧 × 按钮: 删除当前行

### 5. 是否流式输出(单选按钮组)
- **标签**: "是否流式输出"
- **选项**:
  - 是 (空心圆形,未选中)
  - 否 (蓝色填充,当前选中)

### 6. 「回答内容」配置区(可折叠面板)
- **区域标题**: "回答内容",带向下箭头图标
- **帮助图标**: 带圆圈的 i 图标
- **开关组件**: "是否对话历史"(默认关闭)
- **多行文本框**: 
  - 占位符: "可以使用{变量名}的方式引用输出参数中的变量"
  - 支持多行输入
  - 可垂直调整大小

## 技术实现

### 文件修改

1. **EndNode.vue** - 结束节点组件
   - 添加完整的配置界面模板
   - 实现所有交互逻辑
   - 添加详细样式

2. **NodeConfigPanel.vue** - 节点配置面板
   - 为结束节点添加特殊的信息展示区
   - 紫色渐变背景
   - END 徽章标识

3. **LangChainEditor.vue** - 主编辑器
   - 添加 `getAvailableVariables` 函数
   - 为结束节点传递可用变量列表
   - 支持变量引用功能

### 数据结构

```javascript
{
  label: '结束节点',              // 节点名称
  responseMode: 'text',           // 回答模式: text/json/markdown/html
  outputParams: [                 // 输出参数列表
    {
      name: 'result',             // 参数名
      sourceType: 'reference',    // 来源类型: reference/constant/expression
      value: 'node-id.output'     // 参数值
    }
  ],
  streaming: false,               // 是否流式输出
  answerContent: '',              // 回答内容模板
  useHistory: false               // 是否使用对话历史
}
```

## 使用方法

1. **打开配置面板**:
   - 双击画布上的结束节点
   - 或选中节点后按 Ctrl+G

2. **配置回答模式**:
   - 从下拉框选择需要的输出格式

3. **添加输出参数**:
   - 点击输出区域的 + 号按钮
   - 填写参数名
   - 选择参数来源类型
   - 选择或输入具体值

4. **设置流式输出**:
   - 根据需要选择"是"或"否"

5. **编写回答内容**:
   - 在文本框中输入回答模板
   - 使用 `{变量名}` 引用输出参数
   - 例如: `处理结果是: {result}`

6. **保存配置**:
   - 配置自动保存
   - 点击右上角 × 关闭面板

## 特性

✅ **响应式设计** - 适配不同屏幕尺寸  
✅ **实时预览** - 配置即时生效  
✅ **变量引用** - 智能提示可用变量  
✅ **折叠面板** - 可收起不常用的配置项  
✅ **帮助文档** - 每个区域都有帮助图标  
✅ **数据验证** - 确保配置有效性  

## 注意事项

1. 输出参数名建议使用英文,避免特殊字符
2. 引用变量时,确保引用的节点在工作流中位于结束节点之前
3. 流式输出适合长文本场景,可以提升用户体验
4. 回答内容中使用 `{变量名}` 语法引用参数,变量名需与输出参数名一致
