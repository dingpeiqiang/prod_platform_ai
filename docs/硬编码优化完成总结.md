## 使用说明

### 如何调整推荐评分权重

**步骤：**
1. 编辑 `config/system_config.json`
2. 修改 `recommendation` 下的参数
3. 重启服务或调用热加载API

**示例：**
```json
{
  "recommendation": {
    "countScoreWeight": 0.5,      // 提高频次权重
    "userScoreWeight": 0.3,       // 降低用户专属权重
    "timeScoreWeight": 0.2,
    "timeDecayDays": 60            // 延长衰减周期到60天
  }
}
```

### 如何调整字段提取分隔符

**示例：**
```json
{
  "fieldExtraction": {
    "separators": ["是", "为", "：", ":", " ", "="]  // 新增等号分隔符
  }
}
```

### 热加载配置

**API调用：**
```bash
curl -X POST http://localhost:8000/api/v1/config/reload \
  -H "Content-Type: application/json" \
  -d '{"configType": "system_config"}'
```
