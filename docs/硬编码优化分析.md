# 硬编码优化分析报告

## 发现的硬编码问题

### 1. HistoryService 推荐评分参数
**文件：** `app/services/history_service.py`

| 参数 | 当前值 | 说明 | 建议配置化 |
|------|--------|------|-----------|
| `count_score_weight` | 0.4 | 频次评分权重 | ✅ 是 |
| `user_score_weight` | 0.4 | 用户专属评分权重 | ✅ 是 |
| `time_score_weight` | 0.2 | 时间衰减评分权重 | ✅ 是 |
| `count_score_per_unit` | 0.1 | 每出现1次加多少分 | ✅ 是 |
| `user_score_per_unit` | 0.2 | 用户每出现1次加多少分 | ✅ 是 |
| `time_decay_days` | 30 | 时间衰减周期（天） | ✅ 是 |
| `history_query_limit` | 1000 | 查询历史记录条数 | ✅ 是 |
| `recommendation_limit` | 10 | 返回推荐值数量 | ✅ 是 |

### 2. SceneRecognitionSkill 默认场景
**文件：** `app/skills/scene_recognition.py`

| 参数 | 当前值 | 说明 | 建议配置化 |
|------|--------|------|-----------|
| `default_scene` | "generic" | 默认场景 | ✅ 是 |

## 优化方案

### 方案：添加系统配置文件

在 `config/` 目录下新增 `system_config.json`：

```json
{
  "recommendation": {
    "countScoreWeight": 0.4,
    "userScoreWeight": 0.4,
    "timeScoreWeight": 0.2,
    "countScorePerUnit": 0.1,
    "userScorePerUnit": 0.2,
    "timeDecayDays": 30,
    "historyQueryLimit": 1000,
    "recommendationLimit": 10
  },
  "sceneRecognition": {
    "defaultScene": "generic"
  },
  "fieldExtraction": {
    "separators": ["是", "为", "：", ":", " "]
  }
}
```
