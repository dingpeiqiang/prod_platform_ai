# ✅ 历史数据导入模板 - 单条参考数据实现

## 🎯 需求

**用户要求**: "下载的历史数据导入模板中的参考数据太多了，保留1条即可"

### 背景

之前用户下载模板文件时，会得到`import_data_examples`目录中的完整示例文件，可能包含多条数据（如10条）。这会导致：
- ❌ 文件过大
- ❌ 用户需要手动删除多余数据
- ❌ 不够简洁

---

## 🔧 实现方案

### 核心思路

**从示例文件中只提取第一条数据作为模板，添加清晰的注释说明**

```
原始文件 (leave_test_enum.data.jsonl):
  # 注释1
  # 注释2
  {"leave_type":"annual", ...}    ← 第1条
  {"leave_type":"sick", ...}     ← 第2条
  {"leave_type":"personal", ...} ← 第3条
  ... (共10条)
  
  ↓
  
下载的模板文件:
  # 注释1
  # 注释2
  # 以下为参考数据（仅1条，请根据实际业务修改）
  {"leave_type":"annual", ...}    ← 只有第1条
```

#### 修改后

```python
@router.get("/import/template/{formCode}")
async def download_template(formCode: str):
    """
    下载导入模板文件
    
    ⚠️ 重要：只返回第一条数据作为参考，不要太多
    从 import_data_examples 目录读取示例文件，提取第一条数据
    """
    from fastapi.responses import FileResponse
    from io import BytesIO
    
    examples_dir = PROJECT_ROOT / "config" / "import_data_examples"
    template_file = examples_dir / f"{formCode}.data.jsonl"
    
    if not template_file.exists():
        # 返回默认模板
        content = '# 历史数据模板\n# 每行一个JSON对象\n...'
        return FileResponse(...)
    
    # ⚠️ 重要：只读取第一条数据作为模板
    try:
        first_line = None
        comments = []
        with open(template_file, 'r', encoding='utf-8') as f:
            for line in f:
                line = line.strip()
                if not line:
                    continue
                if line.startswith('#'):
                    comments.append(line)
                elif first_line is None:
                    first_line = line
                    break  # ✅ 只取第一条数据
        
        if first_line:
            # 构建模板内容：注释 + 第一条数据
            template_content = '\n'.join(comments[:3]) + '\n'  # 最多保留3行注释
            template_content += f'# 以下为参考数据（仅1条，请根据实际业务修改）\n'
            template_content += first_line + '\n'
            
            logger.info("[config/import/template] 返回模板文件（仅1条数据）: %s", template_file)
            return FileResponse(
                path=BytesIO(template_content.encode('utf-8')),
                media_type='application/jsonl',
                filename=f"{formCode}_template.jsonl"
            )
        else:
            # 文件为空的处理
            ...
    except Exception as e:
        logger.exception("[config/import/template] 读取模板文件失败: %s", e)
        # 错误处理
        ...
```

**改进**:
- ✅ 只提取第一条数据
- ✅ 保留注释说明
- ✅ 添加清晰提示
- ✅ 容错处理

- ✅ 简洁：只保留必要信息
