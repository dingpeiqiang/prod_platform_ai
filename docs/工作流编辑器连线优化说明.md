# 工作流编辑器连线优化说明

## 问题描述
工作流编辑器中节点连接点（port）与连线（connection line）衔接不畅，视觉上存在偏差。

## 优化内容

### 1. 连线坐标计算优化
**文件**: `frontend/src/components/intent-panels/LangChainEditor.vue`

#### 修改前的问题
- 连线起点和终点使用了固定的 `portOffset = 6px` 偏移量
- 导致连线没有精确对准连接点中心
- 贝塞尔曲线控制点计算不够平滑

#### 修改后的改进
```javascript
// 输出端口在节点右侧中间，连接点中心在节点右边缘
const fromX = (fromNode.x + nodeWidth) * scale;
const fromY = (fromNode.y + nodeHeight / 2) * scale;

// 输入端口在节点左侧中间，连接点中心在节点左边缘
const toX = (toNode.x) * scale;
const toY = (toNode.y + nodeHeight / 2) * scale;

// 优化的贝塞尔曲线控制点
const distance = Math.abs(toX - fromX);
const offsetX = Math.max(distance * 0.4, 30); // 最小偏移30px
const fromControlX = fromX + offsetX;
const toControlX = toX - offsetX;
```

**关键改进点**:
1. ✅ 连线起点精确对准节点右边缘（输出端口中心）
2. ✅ 连线终点精确对准节点左边缘（输入端口中心）
3. ✅ 动态调整贝塞尔曲线弯曲度，最小偏移30px确保曲线平滑
4. ✅ 同时修复了正式连线和临时连线（拖拽时）的计算逻辑

## 技术细节

### 坐标系说明
- 节点坐标 `(x, y)` 表示节点左上角位置
- 节点尺寸固定为 `150px × 80px`
- 连接点直径为 `12px`（半径6px）
- 连接点中心位于节点左右边缘的垂直中点

### 贝塞尔曲线算法
使用三次贝塞尔曲线（Cubic Bezier）:
```
M startX startY C control1X control1Y, control2X control2Y, endX endY
```

控制点计算公式:
```javascript
offsetX = max(|endX - startX| × 0.4, 30)
control1X = startX + offsetX
control2X = endX - offsetX
control1Y = startY (保持水平)
control2Y = endY (保持水平)
```

这种设计确保:
1. 曲线在起点和终点处保持水平方向
2. 距离越远曲线弯曲越大
3. 最小偏移30px避免近距离时曲线太平

### 响应式设计
所有坐标计算都乘以 `canvasScale.value`，确保在不同缩放下保持正确的相对位置。
