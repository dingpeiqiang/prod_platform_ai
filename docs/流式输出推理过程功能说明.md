# 流式输出推理过程功能说明

## 概述

聊天助手现在支持**流式输出推理过程**，用户可以实时看到AI的思考过程！

---

## 核心特性

| 特性 | 说明 |
|------|------|
| **Server-Sent Events (SSE)** | 使用标准SSE协议流式输出 |
| **实时渲染** | 前端逐字渲染推理过程 |
| **推理步骤可视化** | 清晰展示思考步骤 |
| **自动切换** | 流式结束后自动显示结果或生成表单 |

---

## 后端API

### POST /api/v1/chat/stream

**流式聊天接口，返回SSE格式**

**请求：**
```json
{
  "messages": [
    {"role": "user", "content": "帮我填一个请假申请"}
  ]
}
```

**响应（SSE流式）：**
```
data: {"type": "thinking", "content": "🔍 正在分析用户输入..."}

data: {"type": "thinking", "content": "📝 用户消息：帮我填一个请假申请"}

data: {"type": "thinking", "content": "🎯 识别到表单需求"}

data: {"type": "thinking", "content": "📋 匹配到表单：请假申请 (leave)"}

data: {"type": "result", "content": "✅ 准备生成表单..."}

data: {"type": "done", "isForm": true}
```

---

## 流式事件类型

| 事件类型 | 说明 | 图标 |
|---------|------|------|
| `thinking` | 思考步骤 | 💭 |
| `result` | 最终结果 | ✅ |
| `error` | 错误 | ⚠️ |
| `done` | 完成（含isForm标识） | - |

---

## 推理步骤内容

### 表单需求示例

```
🔍 正在分析用户输入...
📝 用户消息：帮我填一个请假申请，请假天数3天
🎯 识别到表单需求
📋 匹配到表单：请假申请 (leave)
📄 提取到字段：请假天数
✅ 准备生成表单...
```

### 聊天需求示例

```
🔍 正在分析用户输入...
📝 用户消息：你好
💬 识别为聊天需求
🧠 准备回复内容...
✅ 你好！我是AI智能助手...
```

## 前端实现

### 新增状态

```javascript
const isStreaming = ref(false)
```

### 流式消息类型

```javascript
{
  role: 'assistant',
  type: 'streaming',
  reasoning: [
    { type: 'thinking', content: '...' },
    { type: 'result', content: '...' }
  ],
  timestamp: Date.now()
}
```

### 流式渲染组件

```vue
<div class="streaming-content">
  <div class="reasoning-steps">
    <div v-for="step in msg.reasoning" class="reasoning-step step-type">
      <span class="step-icon">{{ icon }}</span>
      <span class="step-content">{{ step.content }}</span>
    </div>
  </div>
</div>
```
